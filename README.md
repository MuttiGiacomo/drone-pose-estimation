## Build and run


To compile the project, please follow these bash commands

Move into dev folder
~~~
cd dev_ws
~~~

Compile the project
~~~
colcon build
~~~

Install the dependencies
~~~
. install/setup.bash
~~~

To run the simulation, please type
~~~
ros2 launch mds_rover mds_rover_launch.py
~~~
